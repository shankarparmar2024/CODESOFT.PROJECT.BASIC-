## 🎯 About This Repository

This repository contains a collection of **C++ console-based projects** developed during my internship at **CodeSoft**.

The goal of these projects is to:

* Strengthen **core programming fundamentals**
* Build strong **problem-solving skills**
* Gain hands-on experience with **real-world logic implementation**

## ⚙️ Installation & Setup

### 🔽 Clone the Repository

```bash
git clone https://github.com/shankarparmar2024/CODESOFT.PROJECT.BASIC-.git
cd CODESOFT.PROJECT.BASIC-
```

---

### 🛠️ Compile Code

```bash
g++ filename.cpp -o app
```

---

### ▶️ Run Program

#### Windows CMD:

```bash
app
```

#### PowerShell:

```bash
.\app
```
